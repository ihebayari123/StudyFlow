# 📊 Guide d'Utilisation - Questionnaire d'Évaluation

## Comment utiliser le questionnaire ?

### Étape 1 : Démarrer le questionnaire

Vous avez deux options pour démarrer le questionnaire :

1. **Option 1** : Cliquez sur le bouton **"📊 Questionnaire"** dans les réponses rapides
2. **Option 2** : Tapez l'un de ces mots dans le chat :
   - `questionnaire`
   - `évaluation`
   - `test`
   - `analyse`

### Étape 2 : Répondre aux questions

Le système vous posera **5 questions** une par une :

#### Question 1 : Heures de sommeil 😴
```
Combien d'heures dormez-vous par nuit en moyenne ?
Exemple de réponse : 7 ou 7.5
```

#### Question 2 : Heures d'étude 📚
```
Combien d'heures étudiez-vous par jour en moyenne ?
Exemple de réponse : 4 ou 5.5
```

#### Question 3 : Minutes de sport 🏃‍♂️
```
Combien de minutes de sport pratiquez-vous par jour ?
Exemple de réponse : 30 ou 60
```

#### Question 4 : Tasses de café ☕
```
Combien de tasses de café buvez-vous par jour ?
Exemple de réponse : 2 ou 3
```

#### Question 5 : Âge 🎂
```
Quel est votre âge ?
Exemple de réponse : 25
```

### Étape 3 : Consulter vos résultats

Après avoir répondu aux 5 questions, vous recevrez :

#### 📋 Récapitulatif de vos réponses
Toutes vos réponses seront affichées pour vérification.

#### 🧠 Analyse du Stress
- **Score** : Pourcentage de 0 à 100%
- **Niveau** : Bas 🟢 / Normal 🟡 / Modéré 🟠 / Élevé 🔴
- **Conseils personnalisés** selon votre niveau

#### 😊 Analyse de la Joie
- **Score** : Pourcentage de 0 à 100%
- **Niveau** : Bas 😔 / Normal 😐 / Modéré 🙂 / Élevé 😄
- **Conseils personnalisés** selon votre niveau

#### 📊 Visualisation graphique
Des cartes colorées avec des barres de progression pour visualiser vos scores.

## Exemples de Scénarios

### Scénario 1 : Étudiant très stressé

**Réponses :**
- Sommeil : 4 heures
- Étude : 10 heures
- Sport : 10 minutes
- Café : 5 tasses
- Âge : 20 ans

**Résultats attendus :**
- Stress : ~85% (Élevé) 🔴
- Joie : ~25% (Bas) 😔

**Conseils :**
- Augmenter les heures de sommeil
- Réduire les heures d'étude
- Faire plus de sport
- Diminuer la consommation de café

### Scénario 2 : Personne équilibrée

- Étude : 3 heures

- Café : 1 tasse
- Âge : 28 ans

**Résultats attendus :**
- Stress : ~10% (Bas) 🟢
- Joie : ~100% (Élevé) 😄

**Conseils :**
- Continuer les bonnes habitudes
- Maintenir l'équilibre actuel

### Scénario 3 : Travailleur modérément stressé

**Réponses :**
- Sommeil : 6.5 heures
- Étude : 5 heures
- Sport : 30 minutes
- Café : 3 tasses
- Âge : 35 ans

**Résultats attendus :**

**Conseils :**

- Réduire un peu le café

## Conseils pour des Résultats Précis

### ✅ À faire :
- Répondre honnêtement
- Utiliser des moyennes sur une semaine typique
- Utiliser des nombres décimaux si nécessaire (ex: 7.5)
- Prendre le temps de réfléchir à chaque réponse

### ❌ À éviter :
- Donner des réponses extrêmes non réalistes
- Se baser sur une seule journée exceptionnelle
- Entrer des lettres ou des caractères spéciaux
- Répondre trop rapidement sans réflexion

## Validation des Réponses

Le système valide automatiquement vos réponses :

| Question | Format accepté | Plage valide |
|----------|---------------|--------------|
| Sommeil | Nombre décimal | 0 - 24 heures |
| Étude | Nombre décimal | 0 - 24 heures |
| Sport | Nombre entier | 0 - 1440 minutes |
| Café | Nombre entier | 0 - 20 tasses |
| Âge | Nombre entier | 1 - 120 ans |

### Messages d'erreur

Si vous entrez une valeur invalide, vous verrez :
```
❌ Veuillez entrer un nombre valide entre X et Y.
```

Vous pourrez alors corriger votre réponse sans recommencer le questionnaire.

## Refaire le Questionnaire

Vous pouvez refaire le questionnaire autant de fois que vous le souhaitez :

1. Tapez `questionnaire` dans le chat
2. Le système recommencera depuis la question 1
3. Vos anciennes réponses ne seront pas sauvegardées

## Facteurs Clés pour Améliorer vos Scores

### Pour Réduire le Stress :
1. **Dormir 7-9 heures** par nuit
2. **Limiter les heures d'étude** à 4-6h par jour
3. **Faire au moins 60 minutes** de sport par jour
4. **Réduire le café** à 1-2 tasses maximum
5. **Prendre des pauses** régulières

### Pour Augmenter la Joie :
1. **Sommeil optimal** : 7-9 heures
2. **Équilibre travail-vie** : 2-4h d'étude
3. **Activités plaisantes** régulières
4. **Relations sociales** positives
5. **Temps pour soi** chaque jour

## Questions Fréquentes

### Q : Mes données sont-elles sauvegardées ?
**R :** Non, les données sont temporaires et ne sont pas stockées en base de données. Elles sont effacées après l'affichage des résultats.

### Q : Puis-je modifier une réponse ?
**R :** Non, une fois validée, vous devez recommencer le questionnaire. Prenez le temps de bien réfléchir avant de répondre.

### Q : Les résultats sont-ils médicalement valides ?
**R :** Non, ce questionnaire est un outil indicatif. Pour un diagnostic médical, consultez un professionnel de santé.

### Q : Combien de temps prend le questionnaire ?
**R :** Environ 1-2 minutes pour répondre aux 5 questions.

### Q : Puis-je comparer mes résultats dans le temps ?
**R :** Actuellement non, mais cette fonctionnalité est prévue dans une future mise à jour.
