# Studyflow – Stress Management Application

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).  
Studyflow is a web application designed to help students manage stress by organizing tasks, tracking progress, and improving well-being.

## Features
- Create and manage personal study tasks
- Track progress and deadlines
- Stress-reducing tips and reminders
- Dashboard for productivity insights

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- **Symfony** – PHP web framework
- **PHP**
- **MySQL** – Database
- **phpMyAdmin** – Database administration tool

## Architecture
- MVC (Model-View-Controller) architecture with Symfony
- REST API for backend communication
- Frontend interacts via API calls to manage tasks and data
